# webots_ros

[![Build Status](http://build.ros.org/buildStatus/icon?job=Mbin_uB64__webots_ros__ubuntu_bionic_amd64__binary&subject=Noetic)](http://build.ros.org/job/Mbin_uB64__webots_ros__ubuntu_bionic_amd64__binary/)
